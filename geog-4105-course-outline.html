GEOG 4105 – Geospatial Web Programming
Duration: 13 weeks
Format: 3 hours lecture + 3 hours lab per week
Semester: Winter 2026
Course Description

This course prepares students for real-world web GIS development using modern, employer-relevant tools and practices. Students will learn to build interactive, scalable, and responsive geospatial web applications using JavaScript, Python, RESTful APIs, cloud services, and open-source libraries. Emphasis is placed on the design and implementation of web GIS systems that integrate client-side and server-side components, support user interaction, and follow professional standards in UI/UX and web cartography. This course offers students a strong opportunity to develop a portfolio-quality project showcasing full-stack geospatial web development skills applicable to careers in GIS, environmental monitoring, and data-driven digital solutions.

Course Objectives & Program Alignment
  • Work with modern web GIS technologies and frameworks
  • Integrate spatial data into web-based, interactive applications
  • Apply data science and GIS programming skills to build usable tools
  • Strengthen career readiness through a publishable, project-based outcome

Learning Outcomes

  1. Build fully functional, mobile-friendly geospatial web apps
  2. Design and style custom basemaps using Mapbox Studio
  3. Develop and consume REST APIs using Flask and JavaScript
  4. Apply UI/UX and web cartography principles to map design
  5. Plan, wireframe, and scope user-focused GIS applications
  6. Deploy and showcase projects using cloud platforms
  7. Integrate skills from earlier courses into professional-quality applications
  8. Prepare a portfolio-ready project aligned with job roles in web GIS and spatial data science

  • Backend: Python, Flask, GeoJSON, REST APIs
  • Dev Tools: GitHub, VS Code, Render (or Replit/Glitch for beginners)
  • Optional Integration: PostgreSQL/PostGIS or NoSQL (conceptual overview only)
